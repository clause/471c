L3 has the following rules:

- Variable Definition
  - Identifier: a string
  - Let/LetRec: to bind an identifier to a term
  - Reference: to use a variable by its name
- Data Flow Control
  - Immediate: an integer
  - Primitive: binary arithmetic expressions
  - Branch: conditional expressions
- Function
  - Abstract: to define a function with parameters and the function body
  - Apply: to call a function with parameters
- Memory Manipulation
  - Allocate: to allocate a sequence with the given length in memory like an array
  - Load: to load the sequence at the given starting point and length from memory
  - Store: to store a sequence in memory at the given index
